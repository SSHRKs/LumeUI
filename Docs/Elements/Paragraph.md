## Creating Paragraph
```luau
Tab:Paragraph({
    Title = "Paragraph Example",
    Color = "Default",
    Brightness = 28,
})
```

| Tag         | Type        | Default    |
| ----------- | ----------- |------------|
| Title       | String      | "Paragraph"|
| Color       | String      | "Default"  |
| Brightness  | Number      | 28         |
