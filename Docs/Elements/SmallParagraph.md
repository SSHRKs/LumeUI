## Creating Paragraph
```luau
local SmallParagraph = Tab:SmallParagraph({
    Title = "Paragraph Example",
})
```

| Tag         | Type        | Default    |
| ----------- | ----------- |------------|
| Title       | String      | "Paragraph"|
## Set Title
```lua
SmallParagraph:SetTitle('New Title')
```
## Destroy Element
```lua
SmallParagraph:Close()
```
