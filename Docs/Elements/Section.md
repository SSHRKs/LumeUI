## Creating Section
```luau
local Section = Tab:Section({Title = "Toggle Element",})
```

| Tag         | Type        | Default |
| ----------- | ----------- |---------|
| Title       | String      | Section |
| Icon        | String      | nil     |

## Set Title
```lua
Section:SetTitle('New Title')
```
## Destroy Element
```lua
Section:Close()
```
