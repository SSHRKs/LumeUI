## Creating Traffic
```luau
local Traffic = Tab:Traffic({
  Title = "Traffic Title",
})
```

| Tag         | Type        | Default  |
| ----------- | ----------- |----------|
| Title       | String      | nil      |

## Set Title
```luau
Traffic:SetTitle('New Title')
```
## Set Value
```luau
Traffic:SetValue(0)
```
## Reset Traffic
```luau
Traffic:Reset()
```
