## Creating Tab
```luau
local Tab = Window:Tab({
  Title = "Display Elements",
  Icon = "folder", --rbxassetid:// or Lucide Icon
})
```

## Option
| Tag         | Type        | Default     |
| ----------- | ----------- |-------------|
| Title       | String      | Tab         |
| Icon        | String      | nil         |
